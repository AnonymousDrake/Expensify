## Installation

1. Clone the repo :

```
git clone https://github.com/AnonymousDrake/Expensify
```

2. Install packages : _(Use the Yarn package manager in this project. Better to avoid npm)_

```
yarn
```

## Build app :

```
yarn build:dev
```

## Start app :

```
yarn dev-server
```
